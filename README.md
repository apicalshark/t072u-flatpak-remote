# flatpak-remote

Remote builder for a high sea simulator called t072u.

Use at you own risk.

### Using the Remote


```bash
flatpak remote-add --user --if-not-exists torzu oci+https://apicalshark.github.io/t072u-flatpak-remote/
flatpak install onion.torzu_emu.torzu
```
